# dotfiles

Clone repo into the home directory
```
cd ~
git clone https://github.com/dianameca/dotfiles.git
```

Run setup script
```
cd ~/dotfiles
chmod +x setup.sh
./setup.sh
```

Add git user data in a local .gitconfig
```
cat ~/.gitconfig_local
[user]
    name = "..."
    email = "..."
    etc.
```
